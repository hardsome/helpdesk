Должен быть установлен IIS, MS SQL Server (mixed mode)

1. Сохраните папку с проектом там, где IIS сможет получить к ней доступ
(за пределами личных папок), например в папке C:\helpdesk\ 
Выполните скрипт setup.iis7serverapp.vbs
Скрипт создаст веб-приложение для IIS.

2. Выполните скрипты \Server\SQL\DDL.sql и \Server\SQL\TestData.sql 
Скрипты создадут базу данных и заполнят ее данными.

3. Откройте браузер и запустите приложение по адресу
http://localhost/helpdesk/
Если приложение отображается, но данные не отображаются, откройте адрес 
http://localhost/helpdesk/Project/List/ и убедитесь, что сервер вернул JSON данные. 
Устраните проблемы если вместо данных вы видите сообщение об ошибке:

Устранение проблем. 
A. Скорректируйте настройки ODBC подключения в файле  \Server\Utils\config.asp  
Возможно потребуется сменить имя провайдера на SQLNCLI10 или SQLNCLI11 
если SQLNCLI не установлен. 
Возможно потребуется изменить адрес сервера или имя базы данных.
B. Убедитесь, что 404 handler установлен на /Server/Router.asp
С. Убедитесь, что документ по умолчанию: index.html

Чистка.
Когда приложение заработает, удалите файлы setup.iis7serverapp.vbs 
setup.iis6serverapp.vbs
/Server/SQL/*.*
